# SkillCraft Technology - Task 02: Number Guessing Game

A simple command-line game built in Python. The program generates a random number and challenges the user to guess it, providing "too high" or "too low" hints along the way.

## Project Structure
- `src/game.py`: Contains the core logic of the game.
- `main.py`: The main entry point to run the game.

## How to Run

1. Open your terminal or command prompt.
2. Navigate to the root directory of this project:
   ```bash
   cd number-guessing-game
